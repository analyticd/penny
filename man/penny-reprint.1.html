NAME

penny-reprint - read and reprint Penny ledger

SYNOPSIS

penny-reprint FILE...

DESCRIPTION

Reads the Penny ledger(s) you specify and prints them to standard output, in a (hopefully) more tidy format. All comments are retained, but blank lines and insignicant whitespace within the transactions (such as the amount of whitespace between multiple tags) are ignored.

If no FILE, or FILE is -, read standard input.

EXIT STATUS

0 if everything went fine; another value if there were problems (such as inability to read a ledger file.)

BUGS

Please report any bugs in the program or documentation to Omari Norman
